# Gamified Developer Portfolio (2D Interactive)

## Overview
This is an interactive 2D developer portfolio built as a game using JavaScript and the Kaboom.js library.

Instead of a traditional static portfolio, this project allows users to explore a virtual environment (house/map) and interact with different elements to learn more about me, my work, and my background.

This project was initially built by following a YouTube tutorial and then extended with additional custom features and design improvements.

## Features
- Top-down 2D player movement
- Interactive objects with dialogue popups
- Click/touch-based navigation (works on mobile)
- Responsive UI using HTML & CSS
- External links (e.g., resume, projects)
- Camera system following the player
- Custom map design using tile-based approach
